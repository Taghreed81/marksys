# Uniform Solutions Website

[![License: CC BY 3.0](https://img.shields.io/badge/License-CC%20BY%203.0-lightgrey.svg)](https://creativecommons.org/licenses/by/3.0/)

Professional uniform catalog website built on the Stellar template by HTML5 UP.

## Features

- **Responsive Design**: Works on all devices from mobile to desktop
- **Interactive Gallery**: Click-to-zoom image functionality for product viewing
- **Modern UI**: Clean layout with vibrant color scheme
- **Easy Navigation**: Intuitive category browsing and product display
- **Fast Performance**: Optimized assets and lightweight code

## Technologies Used

- HTML5, CSS3, JavaScript
- Stellar template by HTML5 UP (html5up.net/stellar)
- Font Awesome icons
- Pure CSS image zoom system (no JavaScript required)

## Customizations Made

- Implemented interactive product gallery with zoom functionality
- Created responsive category navigation system
- Added professional uniform product displays
- Customized color scheme to match brand identity
- Enhanced mobile experience with touch-friendly elements

## How to Use

1. Clone or download this repository
2. Open `index.html` in any modern browser
3. Browse products by category
4. Click any product image to view zoomed version with details
